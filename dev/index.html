RansacDoneRight.Correspondence — Type
Correspondence{T<:Real} is a pair of points along with their covariances.

Represents the same feature from 2 different views.

\[(x, Σₓ) ↔ (x', Σₓ')\]

RansacDoneRight._apply_homography! — Method

Σₓ contains the covariance matrix of the input.

One may get as: SmallBlockDiagonal(11, UncertainHomography.Σₕ, (@view p.covariance_matrix[1:2, 1:2]))

Used to reduce memory overhead.

RansacDoneRight.compute_identity_statistic — Method

Computes the statistic of 2 image points identity.

Follows a Chi-squared distribution with 2 deg. of freedom (only considers x and y coordinates.)

Two points are identical (p₁ ≡ p₂) when the statistic is close to 0.

source
RansacDoneRight.compute_incidence_statistic — Method

Computes the statistic of incidence of the image point and the line. Assumes the independence of the point and the line.

Follows a Chi-squared distribution with 1 deg. of freedom.

p ∈ l ~ the statistic is close to 0.

Uses the conventions introduced by Forstner: https://www.ipb.uni-bonn.de/pdfs/Forstner2004Uncertainty.pdf

source
RansacDoneRight.compute_inlier_mask — Method

Performs the following test:

H₀: Correspondence is an inlier H₁: Correspondence is an outlier.

mask contains true where H₀ wasn't rejected.

For Reprojection/Forward residual, T is the test statistic corresponding to the quantile of the χ-squared distribution with 4/2 deg. of freedom. 9.49/5.99 corresponds to a confidence level of 0.95.

RansacDoneRight.compute_inlier_test_statistic — Method
compute_inlier_test_statistic(UncertainResidual{Float64})

Uncertain residual is assumed to have a mean at 0 and follow a Multivariate Gaussian distribution.

Calculates the Mahalanobis distance.

One may use Distributions.quantile(_χ²_k_DoF, confidence_level) to get the threshold for the test.

RansacDoneRight.monte_carlo_cov_residuals — Method

Runs the Monte Carlo to estimate the accuracy of First-Order Error Propagation for the covariance of the residuals.

Arguments:

- `truth_correspondences` -- unnoised correspondences
- `run_each_corresp` -- number of runs
- `noise_matrix` -- the 2 × 2 covariance matrix of the points' noise.
- `verbose` -- if `true`, displays large residuals when encountered and adds extra prints for Debug.

Parallelized version (uses Threads).

RansacDoneRight.verify_no_points_are_collinear — Method

Returns true if no points in the minimal set are collinear, and false otherwise.

  • T₁ is the threshold statistic used for 2 points indentity test. Is some quantile(χ²₂). For example T₁=5.99 corresponds to confidence level of 95%.

  • T₂ is the threshold statistic used for point-line incidence test. Is some quantile(χ²₁). For example T₂=3.84 corresponds to confidence level of 95%.

  • check_flip tells whether to check for the flip between the correspondences.
